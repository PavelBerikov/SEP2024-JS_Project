Використовуйте цей додаток для додавання, видалення та сортування даних у форматі ‘ключ = значення’. Ви можете вводити дані кирилицею, латиницею або цифрами. Між ключем і значенням обов’язково має бути знак рівності (=). Щоб виділити елемент для видалення, натисніть на нього — він підсвітиться синім кольором. Після цього натисніть кнопку “Delete”, щоб видалити всі підсвічені елементи.
Use this application to add, delete, and sort data in the format ‘key = value’. You can enter data using Cyrillic, Latin letters, or numbers. The = symbol must be present between the key and the value. To select an item for deletion, click on it — it will be highlighted in blue. Then click the “Delete” button to remove all highlighted items.
